# ffmpeg-installation
A Windows Installation EXE file for FFmpeg

# To use

Just download the EXE file in Release.

# To Complify

1. Download Inno Setup
2. Download FFmpeg from [gyan.dev](https://www.gyan.dev/ffmpeg/builds/) , you need to download "
ffmpeg-release-full-shared.7z" version.
3. Extract the .7z file, place the folder and the ISS file together
4. Complify it.
